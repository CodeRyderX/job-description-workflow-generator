# Job Description Workflow Generator

Converts freelance job descriptions into structured workflow diagrams using AI and a custom DSL.

## ❗ Problem
Freelance applications are time-consuming and unclear. Developers often spend 50–70 minutes breaking down requirements and partially designing systems before even applying.

## 💡 Solution
This tool extracts key requirements from job descriptions and transforms them into Excalidraw-ready workflows, enabling faster understanding and execution.

## 🚀 Results I've seen
- Reduced application time per job from **40–70 minutes → under 10 minutes**
- Faster decision-making on whether to apply
- Clear system visualization before coding and building

## ⚙️ Tech
Node.js, JavaScript, Markdown parsing, Custom DSL, AI prompt engineering, CLI tooling, Excalidraw JSON
